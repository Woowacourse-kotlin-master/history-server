# Commit Convention (AngularJS Style)

| Type | Description |
|------|--------------|
| **feat** | 새로운 기능 추가 |
| **fix** | 버그 수정 |
| **docs** | 문서 수정 (README 등) |
| **style** | 코드 포맷 변경 (세미콜론, 공백 등 — 기능 영향 없음) |
| **refactor** | 코드 리팩토링 (기능 변경 없이 구조 개선) |
| **test** | 테스트 코드 추가 또는 수정 |
| **chore** | 빌드, 설정, 기타 유지보수 작업 |
